如果業主沒有給聲學標準，我們不能假裝沒這件事。

專業的設計師的做法是：先建立 baseline，再把升級路徑說清楚，
這樣設計責任才站得住。

【預留升級結構】

│  Level 1｜基準隔音（Baseline）   
│  • 單層玻璃 STC ≈30            
│  • 一般門                  
│  • 預期 NIC25
│  • 特點：需清楚告知上限         

⬇️

【結構不拆，只多一層玻璃變成雙層玻璃】  

│  Level 2｜中階隔音等級           
│  • 雙層玻璃 STC 40–42 (結構不拆加了一層玻璃)         
│  •一般玻璃門 (門框不留螺絲孔,更換為單層玻璃門扇) 
│  • 預期 NIC 28  
│  • 特點：門為主要瓶頸           

⬇️

【升級玻璃與門】         

Level 3｜專業隔音等級          
│  • 雙層玻璃 STC ≥45(結構不拆將玻璃改為膠合玻璃）            
│  • 隔音門 STC ≥39 (門框不留螺絲孔,更換為雙層玻璃門扇)                
│  • 預期 NIC 30~35       
│  • 特點：需精準施工        

⬇️

【最頂級的隔音】 

│  Level 4｜高保密等級            
│  • 雙層玻璃 STC ≥45            
│  • 隔音門 STC 43(門框不留螺絲孔,更換為雙層玻璃門扇)                  
│  • 預期 NIC 35–40             
│  • 特點：高成本、高責任
