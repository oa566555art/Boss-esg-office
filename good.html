NIC 和 STC 的換算公式
\[ NIC = STC_{eq} - 10\log_{10}\left(\frac{S}{A}\right) \]

STC / NIC / EQ
STCeq = 等效 STC（綜合牆面不同材質的 STC）
S = 牆面面積（平方公尺）
A = 接收房吸音量（Sabin）

① 等效 STC（多段牆體加權）

同一公式支援多段構件（牆、門、玻璃等），以聲能＋面積加權計算整體等效 STC。

NIC聲學的換算公式的STC必須代入等效STC的原因是,因為每一段牆面都有自己的實驗室數值：門、玻璃牆、輕隔間牆各有不同的隔音能力； 當門所在這道牆又有一半是玻璃、另一半是輕隔間牆時，就要把玻璃和輕隔間的隔音數字也納入計算， 再一起加權成一個代表整面牆的單一 dB，這個值就是等效 STC。 現在可直接輸入多段不同構件（牆、門、玻璃、輕隔間等），系統會自動用同一加權公式計算整體等效 STC。

輸入各牆段 STC 與面積後，按「計算多牆段等效 STC」。
\[ TL_{eq}= -10\log_{10}\left( \frac{\sum_{i=1}^{n} S_i10^{-TL_i/10}}{\sum_{i=1}^{n}S_i} \right) \]
重點：等效 STC 不是算術平均；低 STC 且面積較大的構件會明顯拉低整體結果。

② NIC 計算器

依公式計算 NIC（輸入 等效STC~STCeq、牆面積~S、接收房吸音量~A）。

輸入後按「計算 NIC」。
\[ NIC = STC_{eq} - 10\log_{10}\left(\frac{S}{A}\right) \]

若接收房（分貝計）所在位置吸音量來自多種材質，可用吸音係數(α)×面積加總計算： \[ A = \alpha_1 S_1 + \alpha_2 S_2 + \alpha_3 S_3 + \dots \] 隔音係數最佳是1，最差是0，一般方塊地毯（塑膠底背）的吸音係數為0.2~0.3，乘上面積後，就能得出整個接收房的 Sabin 值。例如10㎡的房間只貼地毯，隔音Sabin值為0.2*10㎡=2。 一般辦公室常見： 計算出來之後大約為6～8 dB（視接收空間吸音量多寡而定，也就是說如果想要更高的NIC結果，那麼可以朝增加吸音量的方式來加強）。

③ 目標等效 STC → 反推門 STC

已知目標 STCeq、牆 STC、牆/門面積，反推出「門需要多少 STC」。

輸入後按「反推門 STC」。
若出現無法計算，通常代表：目標 STCeq 過高、或門比例過大、或牆體本身不足以支撐該目標。
